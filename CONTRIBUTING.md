# Contributing

Thanks for your interest! This is an early-stage project, so things move fast and the API may change.

## Getting Started

```bash
cp .env.example .env
# Add OPENROUTER_API_KEY
bun install
bun run dev
```

Requires [Bun](https://bun.sh).

## Making Changes

1. Fork the repo and create a branch
2. Make your changes
3. Open a PR with a clear description of what and why

## What's Welcome

- Bug fixes
- Agent examples in other languages
- Documentation improvements
- Ideas via GitHub Issues

## Questions

Open an issue — happy to discuss before you build something big.
